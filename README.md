# DiscoverDinosaurs
 Simple image classification for 5 types of dinosaurs using fastai.
  1. Ankylosaurus
  2. Brontosaurus
  3. Pterodactyl
  4. T-Rex
  5. Triceratops
 
Dataset : https://www.kaggle.com/datasets/cmglonly/simple-dinosurus-dataset <br />
